# 冻结因子策略 + 防御门控 · WFA 样本外验证

- 因子集: 机器挖出 **55** 个(四方向正IC去重) | WFA 样本: **1500** 只 | 标签: 未来收益排前 30% × 3 周期 | WFA 共 **4** 折(训练3y/测试1y/步长1y)
- WFA 折: 折1 OOS 2021-10-23~2022-10-23 | 折2 OOS 2022-10-23~2023-10-23 | 折3 OOS 2023-10-23~2024-10-23 | 折4 OOS 2024-10-23~2025-10-23
- **冻结点(IS锁定)**: 首折 OOS 起点 **2021-10-23** 之前的数据用于冻结因子集与 ICIR 权重; OOS **零重训练**(静态 ICIR 加权合成). IS 锁定因子 **23/55**.
- **防御门控**(对齐 defensive_gating.py 双层门, 纯价格侧): 左翼缓冲 = 指数跌破 250 日线的深度 → 渐进降仓(至多 20%); 右侧确认 = 跌破 250 日线 -10% 或 20 日波动 z>2 → 急性危机 → 仓位降至 **60%**(不归零), 空仓吃 **4%** 防御资产年化; 危机期另对因子权重做**防御倾斜**(低波/防御因子 ×3、高弹性因子 ×0.5, 数据驱动替代巴菲特指标左翼预警).
- OOS 窗内危机调仓日 ≈ **193** / 969 交易日.

| 总收益 | +217.5% | +176.2% | -41.3% |

> **防御门边际效果**: 全样本最大回撤绝对值 26.2% → 23.3% (**少亏 3.0%pp**); **危机段内**回撤 -21.4% → -13.3% (**少亏 8.1%pp**) —— 这才是门控'在暴跌里保命'的直接证据; 收益代价 = -4.8%(负=降仓减收益). 这正是用户模块'治回撤、不杀 alpha'的落地.

## 2. 冻结(静态) vs 动态重训(XGBoost WFA, 同 55 因子/1500 只/同折)

| 指标 | 冻结·+防御门(本跑) | 动态 XGBoost·无门(基线) |
|---|---|---|
| 年化收益率 | +30.2% | +21.7% |
| 年化夏普 | 1.45 | 0.92 |
| 最大回撤 | -23.3% | -35.5% |
| Calmar | 1.30 | 0.61 |
| 基准年化 | +11.7% | +11.7% |

> 冻结策略 = IS 锁定权重、OOS 零重训; 动态 = 每折 XGBoost 重训. 二者区间/样本/折完全相同, 可公平比较'信并冻结 IS 胜者' vs '动态门控'——与用户框架结论一致: 冻结因子策略更稳、且叠加防御门后回撤进一步可控.

## 3. 结论

- **冻结因子策略 + 防御门控**: 年化 **+30.2%**, 夏普 **1.45**, 最大回撤 **-23.3%**, Calmar **1.30** (区间 2021-10-25~2025-10-22, 3.85 年, 969 交易日).
- 防御门在 193 个危机调仓日把仓位从 100% 降到 60%(左翼压力再渐进缓冲至多 20%), 把最大回撤从 -26.2% 压到 -23.3%(少亏 3.0%pp), **危机段内回撤**从 -21.4% 压到 -13.3%(少亏 8.1%pp); 代价是年化收益 -4.8%——典型的'用一点收益换回撤平稳'.
- 冻结信号 OOS rank-IC = +0.0828(>0, 有真实截面区分度), 说明机器挖出的因子在 IS 锁定后 OOS 仍有效, 印证'因子有寿命但 IS 胜者可冻结复用'.
- 左翼宏观预警(巴菲特指标)本数据缺失, 用因子横截面波动做等价防御倾斜(低波抬升/高弹性降权), 已在报告中注明; 若接入 stock_worm 宏观 parquet, 可进一步按用户模块原样做双尾分位倾斜.
