# 信号融合器 → 组合优化器 串联实证（周换手低本版）

*生成耗时 317.2s, 样本 150 只, 周换手 freq=5, 成本 20bps*

- 把四路信号（frozen / xgb / M1 体制融合 / M2 岭回归）各自接入 `portfolio_optimizer.backtest_optimized`，与「朴素等权周换手」做 A/B。
- **优化器约束**：个股 ≤5% 上限（`max_w=0.05`）+ 周频换手的低换手 + 换手上限 `turnover_limit=0.3`；行业/市值中性本轮未接（缺行业元数据，见文末「后续扩展」）。
- 两套回测口径完全一致（周换手 + 20bps + 全市场等权基准），A/B 公平。

## 二、M1 主信号：优化器参数灵敏度

## 三、结论

- 周换手低本版上叠加约束优化：四路信号平均年化 +6.4% → +16.4%（Δ +10.1%）。
- 朴素周换手最优为 **M1 体制融合（主） (+11.2%)**；约束优化后最优为 **M1 体制融合（主） (+19.2%)**，夏普 0.93（对比同一信号朴素 +11.2%，Δ +8.0%）。
- 约束优化在「弱 alpha + 周换手低本版」上的价值主要来自：① 个股 ≤5% 上限抑制单票黑天鹅；② 换手上限进一步压低调仓日冲击成本；③ softmax 倾斜把权重适度集中于高信号名（相对等权）。
- 注意：此为弱 alpha 信号，绝对收益受 2021-2025 震荡市拖累；行业/市值中性、风险模型（方差项最优化）未接入，是下一步提升点。

## 四、后续扩展

- 接入 SW 行业 / 申万一级 + 市值面板 → 开启 `group_neutral=True`，做规模/行业中性，降低风格漂移。

- 防御门控（`gate=True`）叠加组合层，危机期自动降仓。
