## 因子1: 适度冒险

- **研报标题**: 《成交量激增时刻蕴含的alpha信息——多因子选股系列研究之一》
- **发布日期**: 2022-04（约）
- **计算公式**:
  1. **激增时刻定义**: 剔除开盘收盘，计算每分钟成交量差值（当前分钟-前一分钟），求日内均值mean和标准差std。成交量差值 > mean + std 的分钟定义为"激增时刻"。
  2. **耀眼5分钟**: 激增时刻及其后4分钟共5分钟，称为"耀眼5分钟"。
  3. **月耀眼波动率因子**:
     - 计算每个"耀眼5分钟"内分钟收益率的标准差 → "耀眼波动率"
     - 日内所有耀眼波动率的均值 → "日耀眼波动率"
     - "适度日耀眼波动率" = |日耀眼波动率 - 截面均值|
     - "月均耀眼波动率" = 过去20天"适度日耀眼波动率"的均值
     - "月稳耀眼波动率" = 过去20天"适度日耀眼波动率"的标准差
     - "月耀眼波动率" = (月均耀眼波动率 + 月稳耀眼波动率) / 2（等权合成）
  4. **月耀眼收益率因子**: 同理，用激增时刻的分钟收益率代替波动率，构建"月耀眼收益率"。
  5. **适度冒险因子** = (月耀眼波动率 + 月耀眼收益率) / 2（等权合成）
- **数据需求**: 分钟频OHLCV数据（成交量、收盘价）
- **原始回测**: Rank IC约-6%~-7%，多空年化约20%+
- **来源URL**: https://blog.csdn.net/Menger_Gilmour/article/details/149252467

## 因子2: 完整潮汐

- **研报标题**: 《个股成交量的潮汐变化及"潮汐"因子构建——多因子选股系列研究之二》
- **发布日期**: 2022-05（约）
- **计算公式**:
  1. **邻域成交量**: 每分钟及其前后4分钟（共9分钟）的成交量总和。
  2. **潮汐定义**:
     - "顶峰时刻" = 邻域成交量最高的分钟（第t分钟）
     - "涨潮时刻" = 第5至t-1分钟中邻域成交量最低的分钟（第m分钟），邻域成交量Vm，收盘价Cm
     - "退潮时刻" = 第t+1至233分钟中邻域成交量最低的分钟（第n分钟），邻域成交量Vn，收盘价Cn
  3. **全潮汐因子**: 全潮汐价格变动速率 = (Cn - Cm) / Cm / (n - m)；取过去20天均值。
  4. **强弱拆分**:
     - 若Vm < Vn → 涨潮为"强势半潮汐"，退潮为"弱势半潮汐"
     - 若Vm > Vn → 退潮为"强势半潮汐"，涨潮为"弱势半潮汐"
  5. **强势半潮汐因子**: 强势半潮汐价格变动速率 = (起止收盘价涨跌幅) / (持续分钟数)；取过去20天均值。
  6. **稳定弱势半潮汐因子**: 弱势半潮汐价格变动速率的过去20天标准差。
  7. **完整潮汐因子** = (强势半潮汐 + 稳定弱势半潮汐) / 2（等权合成）
- **数据需求**: 分钟频成交量、分钟收盘价
- **原始回测**: 具体数值未在检索中确认，但被列入综合量价因子之一
- **来源URL**: https://blog.csdn.net/Menger_Gilmour/article/details/149275474

## 因子3: 勇攀高峰（含"灾后重建"因子）

- **研报标题**: 《个股波动率的变动及"勇攀高峰"因子构建——多因子选股系列研究之三》
- **发布日期**: 2022-05-30
- **计算公式**:
  1. **更优波动率**: 第t分钟的更优波动率 = std(t-4至t分钟的OHLC共20个价格数据) / mean(同20个价格数据)，然后取平方。
  2. **收益波动比**: 第t分钟收益波动比 = 第t分钟收益率 / 第t分钟更优波动率
  3. **灾后重建因子**（负向）:
     - 计算每日全天233分钟的收益波动比序列与更优波动率序列的协方差
     - "月均重建" = 过去20天协方差均值
     - "月稳重建" = 过去20天协方差标准差
     - "灾后重建" = (月均重建 + 月稳重建) / 2
  4. **勇攀高峰因子**（正向）:
     - 找到当日更优波动率 >= mean + std 的分钟（异常高波动时段）
     - 计算异常高波动时段的收益波动比与更优波动率的协方差
     - "月均攀登" = 过去20天该协方差均值
     - "月稳攀登" = 过去20天该协方差标准差
     - "勇攀高峰" = (月均攀登 + 月稳攀登) / 2
- **数据需求**: 分钟频OHLC数据（开盘价、最高价、最低价、收盘价）
- **原始回测**:
  - 灾后重建: Rank IC = -6.02%, Rank ICIR ≈ -2.5
  - 勇攀高峰: Rank IC = 5.62%, Rank ICIR = 4.47, 多空年化 = 19.76%, IR = 3.45, 月胜率 = 83.02%
  - 纯净化后: Rank IC = 1.95%, Rank ICIR = 1.61, 多空年化 = 9.04%, IR = 1.52
  - 中证1000: Rank IC = 3.46%, Rank ICIR = 3.35, 多头超额 = 8.67%
- **来源URL**: https://bigquant.com/wiki/doc/AmpLA1MdHN

## 因子4: 球队硬币

- **研报标题**: 《个股动量效应的识别及"球队硬币"因子构建——多因子选股系列研究之四》
- **发布日期**: 2022-06-11
- **计算公式**:
  - 核心理论: Moskowitz(2021) "可知性"概念——硬币型（高可知性→动量预期→实际反转）vs 球队型（低可知性→反转预期→实际动量）
  - 识别标准: 波动率低 + 换手率下降 → "硬币型"（×(-1)翻转）；波动率高 + 换手率上升 → "球队型"（保持不变）
  - **三维度分解**: 日间反转、日内反转、隔夜反转
  - **日间反转-波动翻转**: 日间收益率 = 日度收益率；过去20天均值和标准差；波动率 < 截面均值 → ×(-1)
  - **日间反转-换手翻转**: 换手率变化量 = t日换手率 - (t-1)日换手率；> 市场均值的为"球队型"，< 市场均值的为"硬币型"（×(-1)）
  - **日内反转**: 同理，日内收益率 = 收盘价/开盘价 - 1
  - **隔夜反转**: 隔夜收益率 = t日开盘价/(t-1)日收盘价 - 1；引入"均值距离化": |隔夜收益率 - 市场平均隔夜收益率|
  - **合成**: 修正日间反转 = (波动翻转 + 换手翻转) / 2；修正日内反转 = 同理；修正隔夜反转 = 同理
  - **球队硬币因子** = (修正日间反转 + 修正日内反转 + 修正隔夜反转) / 3
- **数据需求**: 日频开盘价、收盘价、换手率（或自由流通换手率）
- **原始回测**:
  - Rank IC = -9.67%, Rank ICIR = -4.73, 多空年化 = 39.69%, IR = 3.95, 月胜率 = 85.14%
  - 纯净化后: Rank IC = -4.35%, Rank ICIR = -2.98, 多空年化 = 22.27%, IR = 2.75
  - 沪深300/中证500/中证1000多头超额: 5.86% / 12.12% / 16.55%
- **来源URL**: https://bigquant.com/wiki/doc/8Hnv2QjnMp , https://zhuanlan.zhihu.com/p/645517799

## 因子5: 云开雾散

- **研报标题**: 《波动率的波动率与投资者模糊性厌恶——多因子选股系列研究之五》
- **发布日期**: 2022-08-04
- **计算公式**:
  1. **模糊性定义**（波动率的波动率）:
     - 分钟收益率: r(t) = C(t)/C(t-1) - 1
     - 第t分钟波动率: vol(t) = std(r(t-4)至r(t))（5分钟窗口）
     - 第t分钟模糊性: ambig(t) = std(vol(t-4)至vol(t))（5分钟窗口的波动率的标准差）
  2. **"模糊关联度"因子**: 每日模糊性序列与分钟成交金额序列的相关系数；"均模糊关联度" = 过去20天均值，"稳模糊关联度" = 过去20天标准差，等权合成。
  3. **"模糊金额比"因子**:
     - "起雾时刻" = 模糊性 > 当日均值的分钟
     - "雾中金额" = 起雾时刻的分钟成交金额均值
     - "总体金额" = 日内所有分钟成交金额均值
     - "日模糊金额比" = 雾中金额 / 总体金额
     - "均/稳模糊金额比" = 过去20天均值/标准差，等权合成
  4. **"模糊数量比"因子**: 同上，用成交量代替成交金额。
  5. **"修正模糊价差"因子**:
     - "日模糊价差" = 日模糊金额比 - 日模糊数量比（反映急于卖出时的流动性成本）
     - 对日模糊价差 < 0 的部分，用过去10天标准差调整（识别基本面恶化vs反应过度）
     - "均修正模糊价差" = 过去20天均值
     - "修正模糊价差" = (均修正模糊价差 + 稳模糊价差) / 2
  6. **云开雾散因子** = (模糊关联度 + 模糊金额比 + 修正模糊价差) / 3（等权合成）
- **数据需求**: 分钟频收盘价、成交量、成交金额
- **原始回测**:
  - Rank IC = -9.81%, Rank ICIR = -4.48, 多空年化 = 30.89%, IR = 3.29, 月胜率 = 82.46%
  - 纯净化后: Rank IC = -3.95%, Rank ICIR = -3.22, 多空年化 = 13.38%, IR = 2.17
  - 沪深300/中证500/中证1000多头超额: 5.75% / 6.39% / 10.59%
- **来源URL**: https://bigquant.com/wiki/doc/OeSNIkQANe

## 因子6: 飞蛾扑火

- **研报标题**: 《个股股价跳跃及其对振幅因子的改进——多因子选股系列研究之六》
- **发布日期**: 2022-09-22
- **计算公式**:
  1. **月跳跃度因子**:
     - 剔除开盘收盘，仅考虑日内分钟数据
     - 单利收益率: r_simple(t) = C(t)/C(t-1) - 1
     - 连续复利收益率: r_log(t) = ln(C(t)/C(t-1))
     - 单复利差: diff(t) = r_simple(t) - r_log(t)
     - 泰勒残项: taylor(t) = diff(t) × 2 - r_log(t)²
     - 日跳跃度 = 日内所有泰勒残项的均值
     - 月跳跃度 = 过去20个交易日的日跳跃度均值
  2. **修正振幅因子**: 使用日频最高价和最低价数据，对传统振幅因子进行反转修复（具体修正方式为剥离跳跃部分后的残差振幅）
  3. **飞蛾扑火因子** = (月跳跃度 + 修正振幅) 合成
- **数据需求**: 分钟频收盘价、日频最高价最低价
- **原始回测**: Rank IC = -8.90%, Rank ICIR = -4.52, 多空年化 = 37.30%, IR = 3.51
- **来源URL**: https://blog.csdn.net/weixin_42219751/article/details/127451380

## 因子7: 草木皆兵

- **研报标题**: 《显著效应、极端收益扭曲决策权重和"草木皆兵"因子——多因子选股系列研究之八》
- **发布日期**: 2022-12-13
- **计算公式**:
  1. **原始惊恐因子**:
     - 将每日收益率视为投资者决策权重
     - 每天收益率偏离市场的程度 = |r(t) - r_market(t)| 作为极端收益对决策权重的扭曲程度
     - 使用该偏离程度加权每日收益率，模拟投资者决策过程
     - 原始惊恐 = Σ(偏离程度 × 日收益率) / Σ(偏离程度)
  2. **改进方向**: 基于学术研究结论（短期反转效应、小市值更强烈、波动率加剧时更强烈），从三个角度改进：
     - 波动率加剧角度
     - 个人投资者交易比角度
     - 注意力衰减角度
  3. **草木皆兵因子**: 改进后的最终因子（负向因子）
- **数据需求**: 日频收益率、市场收益率、波动率、个人投资者交易比等
- **原始回测**:
  - Rank IC = -8.90%, Rank ICIR = -4.54, 多空年化 = 32.50%, IR = 3.92, 月胜率 = 85.71%
  - 纯净化后: Rank IC = -3.59%, Rank ICIR = -1.86, 多空年化 = 16.40%, IR = 2.03
  - 沪深300/中证500/中证1000多头超额: 7.42% / 9.74% / 10.15%
- **来源URL**: https://finance.sina.com.cn/stock/stockzmt/2022-12-14/doc-imxwqxpc9535558.shtml

## 因子8: 水中行舟（含"随波逐流"和"孤雁出群"子因子）

- **研报标题**: 《个股成交额的市场跟随性与"水中行舟"因子——多因子选股系列研究之九》
- **发布日期**: 2023-02-15
- **计算公式**:
  1. **随波逐流因子**（正向，考察价格高位时成交额与市场的关联性）:
     - "合理收益率" = 过去20天日内收益率（收盘价/开盘价-1）的均值
     - "相对开盘收益率" = 每分钟收盘价/当日开盘价 - 1
     - "高位成交额" = 相对开盘收益率 > 合理收益率的分钟成交额之和
     - "低位成交额" = 相对开盘收益率 < 合理收益率的分钟成交额之和
     - "高低额差" = (高位成交额 - 低位成交额) / 当日流通市值
     - 计算每只股票过去20天高低额差序列与其余所有股票的Spearman相关系数的绝对值的均值
  2. **孤雁出群因子**（负向，考察市场分化不明显时成交额的独立性）:
     - "分钟市场分化度" = 每一分钟所有股票分钟收益率的标准差
     - "不分化时刻" = 分钟市场分化度 < 当日均值的分钟
     - 计算不分化时刻每只股票的分钟成交额序列与其余股票的Pearson相关系数的绝对值的均值
     - "月均/月稳孤雁出群" = 过去20天均值/标准差，等权合成
  3. **水中行舟因子** = (随波逐流 + 孤雁出群) / 2（等权合成）
- **数据需求**: 分钟频收盘价、开盘价、成交额、日频流通市值
- **原始回测**:
  - 随波逐流: Rank IC = 5.89%, Rank ICIR = 3.29, 多空年化 = 16.26%
  - 孤雁出群: Rank IC = -8.20%, Rank ICIR = -3.55, 多空年化 = 31.26%
  - 水中行舟: Rank IC = -9.36%, Rank ICIR = -4.95, 多空年化 = 36.24%, IR = 4.40, 月胜率 = 86.67%
  - 纯净化后: Rank IC = -4.71%, Rank ICIR = -4.36, 多空年化 = 17.59%, IR = 3.34
  - 沪深300/中证500/中证1000多头超额: 9.48% / 9.71% / 18.51%
  - 周频调仓效果更佳: 多头年化 = 33.72%, 多空年化 = 60.22%
- **来源URL**: https://bigquant.com/wiki/doc/uJ2mq8I9A1

## 因子9: 花隐林间（含"朝没晨雾"、"午蔽古木"、"夜眠霜露"子因子）

- **研报标题**: 《推动个股价格变化的因素分解与"花隐林间"因子——多因子选股系列研究之十》
- **发布日期**: 2023-03-27
- **计算公式**:
  - 核心逻辑: 将推动个股价格变化的因素分解为市场层面推动力、个股突发信息、中长期基本面信息、噪声四类。前三类和噪声的推动力越小，股票越可能被低估，未来收益越高。
  - **回归模型**: 对每分钟的过去5分钟成交量与当前分钟收益率做线性回归，获得回归系数权重w及其t值，同时计算模型F值和截距t值。
  1. **朝没晨雾因子**:
     - 计算回归得到的t-1至t-5分钟的5个t值的标准差
     - 标准差越小 → 短时间内流入的信息越平稳 → 越没有突发信息
     - "朝没晨雾" = 过去20天"日朝没晨雾"（t值标准差）的均值
  2. **午蔽古木因子**: （基于回归F值或截距t值构建，逻辑类似）
  3. **夜眠霜露因子**: （基于回归相关指标构建，逻辑类似）
  4. **花隐林间因子** = 朝没晨雾 + 午蔽古木 - 夜眠霜露

## 因子16: 滴水穿石（用户记忆中的"水滴石穿"）

- **研报标题**: 《个股日内成交量周期性节奏刻画与"滴水穿石"因子构建——多因子选股系列研究之二十四》
- **发布日期**: 2025-12-16
- **计算公式**:
  1. **数据清理**: 剔除集合竞价交易数据（开盘1分钟、收盘前3分钟）
  2. **去脉冲处理（IQR限幅）**: 对日内成交量序列做IQR限幅
     - IQR = 75%分位数 - 25%分位数
     - 下限 = 中位数 - 3×IQR，上限 = 中位数 + 3×IQR
     - vol_clipped = clip(vol, lower, upper)
  3. **去均值+Hann窗**: xw(t) = (vol_clipped(t) - mean(vol_clipped)) × w(t)，其中w(t)为Hann窗
  4. **计算rFFT与功率谱**:
     - FFTc(f) = rfft(xw(t))
     - P(f) = |FFTc(f)|²
  5. **提取目标频带能量**:
     - 频谱到周期转换: period(f) = 1 / freq(f)
     - 240分钟数据经rFFT得到120个频域点，每个点对应频率1/240
     - 2分钟对应周期0.5（第48个点），5分钟对应周期0.2（第120个点）
     - band_power = Σ P(f) for f in [48, 120]（2-5分钟周期的能量）
     - total_power = Σ P(f) for f in [1, 120]（总能量，排除直流分量f=0）
  6. **滴水穿石因子** = band_power / (total_power + ε)
  - **Python代码核心**:
    ```python
    def cal_factors(vol):
        q = vol.quantile([0.25, 0.75])
        med = q.median()
        iqr = q.iloc[-1] - q.iloc[0]
        vol = vol.clip(lower=med-3*iqr, upper=med+3*iqr)
        vol = vol - vol.mean()
        vol = vol * np.hanning(len(vol))
        fft_cf = np.fft.rfft(vol)
        p_f = np.square(np.abs(fft_cf))
        return np.sum(p_f[48:]) / np.sum(p_f[1:])
    ```
- **数据需求**: 分钟频成交量数据（1分钟K线）
- **原始回测**:
  - 6年IC超0.1，其中3年IC超0.12
  - 五年累计IC突破100
  - 多空年化 = 42%，多头年化 = 20%
  - 市值+行业中性化后: 五年累计IC ≈ 80+, 多头年化 = 18%, 多空年化 = 35%
  - 进一步剥离流动性因子后: 五年累计IC ≈ 70, 多头年化 = 20%, 多空年化 = 35%
  - 全风格中性化后: 五年累计IC ≈ 40, 多头年化 = 12%, 多空年化 = 20%
- **来源URL**: https://zhuanlan.zhihu.com/p/1986126096874366670 , https://zhuanlan.zhihu.com/p/1985628109588948000 , https://bigquant.com/wiki/doc/ubHzz53Mtp

## 因子A: 聪明钱（Smart Money）

- **研报标题**: 《"聆听高频世界的声音"系列研究（三）：跟踪聪明钱，从分钟行情数据到选股因子》
- **发布日期**: 2016-07-08
- **分析师**: 魏建榕、高子剑
- **计算公式**:
  1. **聪明度指标S**: S = |R| / √V
     - R = 分钟涨跌幅（绝对值）
     - V = 分钟成交量
  2. **聪明钱筛选**: 对特定股票、特定时段的所有分钟行情数据，按S从大到小排序，取成交量累积占比前20%的分钟视为"聪明钱"交易
  3. **聪明钱情绪因子Q**: 反映聪明钱参与交易的相对价位
     - Q = 聪明钱交易的VWAP / 全部交易的VWAP
     - Q > 1 → 聪明钱在高位买入 → 看跌信号
     - Q < 1 → 聪明钱在低位买入 → 看涨信号
  4. **2.0版本改进**（2020年）:
     - S = |R| / (V^β)，β为可变参数
     - 原始β=0.5，最优β≈0.1
     - 改进方案S3: S = |R| / ln(V)
     - 改进后IC均值 = -0.050, 多空IR = 3.74（原始IR = 1.69）
- **数据需求**: 分钟频OHLCV数据
- **原始回测**: 回测区间2013.04-2019.12，月度调仓，IR = 1.69（原始），3.74（改进S3）
- **来源URL**: https://zhuanlan.zhihu.com/p/146320622 , https://bigquant.com/wiki/doc/V8A7T4hGwJ
