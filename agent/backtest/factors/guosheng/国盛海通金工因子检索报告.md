# 国盛 / 海通证券金融工程因子检索报告

> 复现自国盛金工『量价淘金』系列与海通金工『选股因子系列』研究。
> 选取与现有因子库(反转/量价/微观结构/波动率/资金流)正交、且在沙箱 5m/日频 OHLCV 上可复现的 3 个因子。

## 复现因子清单

### 1. 隔夜收益 `overnight_return` — 国盛金工『量价淘金 / 隔夜因子』研究

- **公式**:隔夜收益 = 今开 / 昨收 − 1, 取过去 20 日均值。捕捉隔夜信息(业绩/新闻/美股)的跳空定价。
- **逻辑**:国盛研究发现传统隔夜收益(弱动量)经变换后是有效选股因子。本实现取原始隔夜收益均值。
- **数据**:日频 open/close(由 5m resample; open=当日首根 5m, 昨收=前交易日末根 5m)。前一日取前一交易日收盘,避免周末跳空误算。
- **复现结果**:IC=+0.022, ICIR=+1.42, 多空夏普 +0.69(做多高值方向有效)。

### 2. 量价背离 `volume_price_divergence` — 国盛金工『量价淘金』系列

- **公式**:日收益与日成交量滚动(20 日)相关系数。高相关=放量确认趋势;低/负=量价背离(警示)。
- **逻辑**:国盛『量价淘金』核心——同一交易行为(量价关系)多视角刻画。
- **复现结果**:IC≈0, ICIR=+0.17, **多空夏普 +1.41**(全样本最强新因子, 但 IC 弱, 极值分组捕捉到非单调关系, 上线前须 walk-forward 验证)。

### 3. Amihud 非流动性 `amihud_illiquidity` — 海通金工『选股因子系列』(流动性方向)

- **公式**:ILLIQ = (1/D) Σ |r_t| / V_t, V_t 为日成交金额。高非流动性=缺深度, 有流动性溢价(正向因子)。
- **逻辑**:学术最常用流动性代理(Amihud 2002)。沙箱 5m 无 `amount`, 以 volume 代理 V_t。
- **复现结果**:IC=−0.005, ICIR=−0.35, 多空夏普 −0.87(取反后 0.87)。
