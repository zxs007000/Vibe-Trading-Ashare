## 0. 主线 Thesis 与项目背景

整个项目的出发点不是「再找一个无敌因子」，而是回答一句更务实的话：

> **因子有寿命。在 IS（样本内）有效的因子，到 OOS（样本外）未必还有效；在牛市有效的因子，在熊市未必还有效。所以真正该做的是——判断当前处于什么状态，启用该状态下还活着的因子。**

落地成两条可执行的工程约束：

1. **选股层（因子维度）**：把因子集与权重在 IS 一次性「冻结」，OOS 零重学习。已用严格 OOS 验证证明，冻结 IS 胜者并不劣于持续自适应门控——一次性好选择够用。
2. **配置层（市场状态维度）**：用 regime 信号（宽基趋势 / 波动率状态）做总闸，risk-on 持股票组合、risk-off 切国债。但本次复盘**新增核心约束**：regime 信号的尺度必须与 alpha 的再平衡尺度匹配（详见 §6.2、§7）。

## 2. 方法论

### 2.1 严格 OOS 框架

- **分界点 `SPLIT = 2024-09-01`**：A股「924 政策行情」regime 切换（三年熊市→急涨），经济含义强。
- **IS（样本内）= 2006 ~ 2024-08**：锁定因子集与权重。
- **OOS（样本外）= 2024-09 起**（约 1.75 年）：零重学习地应用锁定结果。
- 2020–2023 跨周期检验用**另一套冻结切点 `FREEZE_CUT = 2020-01-01`**（用 2020 之前数据冻结，在 2020–2023 严格 OOS）。

### 2.2 关键参数（来自 `oos_validation.py`）

| 参数 | 值 | 含义 |
|---|---|---|
| `TOP_K` | 0.30 | 每期取截面排名前 30% |
| `HOLD` | 5 | 非重叠 5 交易日（约一周）再平衡 |
| `COST` | 0.001 | 单边千一成本（组合层面 `top_k*2*cost` 近似） |
| `TRAIL` | 250 | 滚动 IC 统计窗口（≈1 年） |
| `SPLIT` | 2024-09-01 | 严格 OOS 分界 |
| `RNG` | 42 | 随机种子 |
| `FWD_HORIZON` | 5 | 用 5 日前向收益算 rank-IC |

> 注意：`HOLD=5`（约一周），不是此前对话里误发的「20 日」。配置层 regime 信号的尺度问题正是围绕这个 5 日再平衡尺度展开的（见 §6.2、§7）。

### 2.3 因子预处理（与回测框架一致）

1. **横截面 z-score 标准化**：每天把全市场该因子分布拉成均值 0、标准差 1，使不同股票可比。
2. **证监会行业中性化**（`neutralize_factors`）：每个交易日，对每只股票减去其所在行业的横截面均值，剔除「行业整体在涨」的干扰。质量因子额外做**规模 10 分桶 demean + 行业 demean**。
3. 正交族因子在代入前也统一 float32 并去极端无穷。

### 2.4 信号合成（`build_signal`）

每只股票的综合分：

```
composite = Σ (orient(f) × w(f) × z(f)) / Σ w(f)
```

- `orient(f) = +1 if IS_mean_IC(f) ≥ 0 else -1`：**按 IS 期的 IC 符号自动翻转方向**。IC 为负的因子（如多数动量/波动率因子）自动反向取，低分反而加分。
- `w(f) = is_icir(f)`（Frozen 模式 `weight_src="is"`）：权重 = IS 期 ICIR（信息比率），越稳的因子权重越大。
- `gate=False`（Frozen）：连「最近不灵就停」都不做，OOS 永远全开冻结集。

### 2.5 选股

每 `HOLD=5` 交易日，取综合分 `top-K`（前 30%）等权持有；组合日收益 = 持有股前向收益均值 − 成本。

### 2.6 「冻结集」定义

```
frozen_set = { f | IS_mean_IC(f) > 0 且 IS_ICIR(f) > 0 }
```

- 全样本冻结（IS ≤ 2024-09-01，38 因子池）：**进 Frozen 16/38**。
- 2020–2023 冻结（pre-2020，38 因子池）：**进 Frozen 14/38**。

## 3. 因子家族总览（38 因子 · 6 族）

因子池 = **30 技术 + 3 基本面质量 + 5 状态正交 = 38 因子**，按下述 6 族组织（与 `factor_window_check.py` 的 `FAMILY` 映射一致）。

| 家族 | 因子数 | 全样本进 Frozen | 2020–2023 窗口内 活/死 | 2020–2023 均 ICIR | 经济含义 |
|---|---|---|---|---|---|
| **反转** | 6 | 4/6（rev_5/20/60/intraday） | 4 活 / 2 死 | **+1.07** | 短期超跌反弹（均值回复） |
| **动量** | 6 | 0/6 | 0 活 / 6 死 | −2.05 | 追涨（A股短/中周期失效） |
| **波动** | 7 | 1/7（drawup_60） | 1 活 / 6 死 | −6.59 | 波动率 / 风险（多为负向异象） |
| **流动性** | 11 | 2/11（overnight_gap、amihud_20） | 2 活 / 9 死 | −2.87 | 微结构 / 量价 / 流动性 |

## 4. 因子公式详解（核心）

> 以下公式均取自源码 `factor_zoo_daily.py`（`build_factors`）、`factor_zoo_ortho.py`（`build_ortho_factors`）、`build_fundamental_factors.py`。
> 通用记号：`close/open/high/low/vol/amount` 为日线；`ret = close.pct_change()`（日收益）；`mkt = ret.mean(axis=1)`（等权市场日收益，与 regime 信号同源、无未来泄漏）。

### 4.1 反转族（6 个）— 短期均值回复

| 因子 | 公式 | 说明 |
|---|---|---|
| `rev_5` | `-(close/close.shift(5) − 1)` | 过去 5 天收益取反；跌多→值大 |
| `rev_20` | `-(close/close.shift(20) − 1)` | 过去 20 天收益取反 |
| `rev_60` | `-(close/close.shift(60) − 1)` | 过去 60 天收益取反 |
| `rev_intraday` | `-(close − open)/open` | 日内（收盘相对开盘）收益取反 |
| `mom_5` | `close/close.shift(5) − 1` | 名义动量；**实测 IC 为负 → 当反转用** |
| `mom_20` | `close/close.shift(20) − 1` | 名义动量；**实测 IC 为负 → 当反转用** |

> 结构本质：前 4 个 = 「动量取负号」。逻辑是短期跌多了会反弹，所以因子值大（跌多）→ 未来涨 → 买入。后两个名义是动量，但在 A 股短周期 IC 为负，引擎按 IS-IC 符号自动翻转（`orient=-1`），行为与反转一致，故并入反转族计权重。

### 4.2 动量族（6 个）— 追涨（本池基本失效）

| 因子 | 公式 | 说明 |
|---|---|---|
| `mom_60` | `close/close.shift(60) − 1` | 60 日动量 |
| `mom_120` | `close/close.shift(120) − 1` | 120 日动量 |
| `mom_250` | `close/close.shift(250) − 1` | 250 日动量（年度） |
| `mom_12_1` | `close.shift(21)/close.shift(252) − 1` | Carhart 12-1 月（跳过最近 1 月） |
| `ma_dev_20` | `close/ma20 − 1`，`ma20=close.rolling(20).mean()` | 偏离 20 日均线 |
| `ma_dev_60` | `close/ma60 − 1`，`ma60=close.rolling(60).mean()` | 偏离 60 日均线 |

> 2020–2023 窗口内该族 **0/6 活**，均 ICIR −2.05：新冠后趋势策略崩溃，中长周期动量在 A 股整体反向。

### 4.3 波动族（7 个）— 波动率 / 风险

| 因子 | 公式 | 说明 |
|---|---|---|
| `vol_20` | `ret.rolling(20).std()` | 20 日收益波动 |
| `vol_60` | `ret.rolling(60).std()` | 60 日收益波动 |
| `ret_skew_60` | `ret.rolling(60).skew()` | 60 日收益偏度 |
| `ivol_60` | `resid.rolling(60).std()`，`resid = ret − beta_60·mkt` | 特质波动（剔除市场 beta 的残差波动） |
| `downside_vol_60` | `ret.clip(upper=0).rolling(60).std()` | 仅下行波动 |
| `boll_w` | `(ma20+2·s20 − (ma20−2·s20))/ma20 = 4·s20/ma20` | 布林带宽（s20=20日标准差） |
| `drawup_60` | `close.rolling(60).max()/close − 1` | 距 60 日高点的回撤（回撤近似） |

> 2020–2023 窗口内该族 **1/6 活**（仅 drawup_60 进 Frozen），均 ICIR −6.59：波动率类因子整体强烈负向（高波动股未来跑输，即低波溢价反向表达）。

### 4.4 流动性 / 微结构 / 量价族（11 个）

| 因子 | 公式 | 说明 |
|---|---|---|
| `amihud_20` | `(ret.abs()/amount).rolling(20).mean()` | Amihud 非流动性（越高越不流动） |
| `dolvol_trend` | `amount/amount.rolling(20).mean() − 1` | 近期流动性 vs  trailing |
| `macd_hist` | `(macd − macd.ewm(9))/close`，`macd=ema12−ema26` | 标准化 MACD 柱 |
| `rsi_14` | `1 − 1/(1 + gain.mean(14)/loss.mean(14))` | RSI（0~1） |
| `adx_14` | 标准 ADX（趋向强度，14） | 趋势强弱（向量化实现） |
| `high_52w` | `close/close.rolling(252).max() − 1` | 距 52 周新高 |
| `overnight_gap` | `open/close.shift(1) − 1` | 今开 / 昨收（隔夜跳空） |
| `intraday_range` | `(high − low)/close` | 日内振幅 |
| `vol_ratio` | `vol/vol.rolling(20).mean()` | 异常放量 |
| `vol_price_corr` | `ret.rolling(20).corr(vol.pct_change())` | 量价相关性 |
| `amount_strength` | `amount/amount.rolling(60).mean() − 1` | 成交强度 |

> 该族在 `factor_window_check` 的 `FAMILY` 映射里统一归为「流动性」。2020–2023 窗口内 **2/9 活**（overnight_gap、amihud_20），均 ICIR −2.87。

### 4.5 质量族（3 个）— 基本面质量（最稳）

数据源：财报（akshare `stock_financial_abstract` / stock_worm 季报）→ 长表 → **前向填充到日线日历** → **规模 10 分桶 demean + 证监会行业 demean** → 截面 z-score，存于 `fund_factors_daily.parquet`，回测时直接加载。

| 因子 | 含义 | 公式（财报口径） |
|---|---|---|
| `ROE` | 净资产收益率 | 净利润 / 净资产（WEIGHTAVG_ROE） |
| `rev_yoy` | 营业总收入同比增长率 | (本期营收 − 去年同期营收) / 去年同期营收 |
| `profit_yoy` | 归属母公司净利润同比增长率 | (本期归母净利 − 去年同期) / 去年同期 |

> 三个质量因子 **全部进 Frozen（3/3）**，作「恒定正交质量倾斜」而非状态开关；是慢而稳的正信号。2020–2023 窗口内 **3/3 全活、均 ICIR +3.52（全族最高最稳）**。

### 4.6 正交防御族（5 个）— 状态正交

经济动机：构造理论上「在一种市场状态出生、在另一种状态死亡」的因子，让「什么状态用什么因子」在选股层真正生效。β 用显式滚动协方差公式向量化计算。

| 因子 | 公式 | 经济含义 / 偏态 |
|---|---|---|
| `beta_60` | `cov(ret, mkt)_60 / var(mkt)_60` | 60 日滚动 beta 至等权市场 → 偏牛 |
| `lowvol_60` | `−ret.rolling(60).std()` | 低波防御 → 偏熊 |
| `lowivol_60` | `−resid.rolling(60).std()`，`resid=ret−beta_60·mkt` | 低特质波动（控住 beta）→ 偏熊 |
| `liq_stress_20` | `amihud_20 / amihud_20.rolling(60).mean() − 1` | 流动性压力（流动性正在枯竭）→ 偏熊 |
| `distress_60` | `ret.clip(upper=0).rolling(60).std()` | 困境下行波动 → 偏熊 |

> 实测结论（见 §6.1 verdict）：`beta_60` 的 IC≈0（市场 beta 不是 A 股截面选股因子），低波/流动性压力两态皆活（同质），困境两态皆死。故「选股层状态开关」这条路被证伪，资源收敛到 Frozen + 质量层 + 配置总闸。但在 2020–2023 窗口，正交族 **4/5 活、均 ICIR +2.12**，是仅次于质量的最稳族（低波溢价在熊市/危机段最强）。

## 5. 反转因子的结构（专门展开）

用户重点问的「你写的那几个反转因子」结构如下：

### 5.1 反转族由「4 个真反转 + 2 个伪装反转」组成

- **4 个真反转**：`rev_5 / rev_20 / rev_60 / rev_intraday`，本质是「动量取负号」。
- **2 个伪装反转**：`mom_5 / mom_20`，名义是动量，但 A 股短周期 IC 为负，引擎按 IS-IC 符号自动翻转（`orient=-1`），实际行为与反转一致，故并入反转族计权重。

### 5.2 结构核心：那个「负号」

```
rev_5 = -(close/close.shift(5) - 1)
```

1. 原始动量 = 「过去涨了多少」；加负号 → 变成「过去跌了多少」。
2. 逻辑链：短期跌多了 → 情绪超跌 → 均值回复反弹 → 因子值大（跌多）→ 未来涨 → 买入。
3. `rev_intraday = -(close-open)/open` 不是「N 日」反转，而是**日内反转**：今天收盘高于开盘（日内涨）→ 因子为负 → 预期次日回落。背后是「隔夜跳空 + 日内过度反应」微结构异象，与前三个按 N 日算的不是一回事。

### 5.3 算完还不能直接买卖：还要过两道

1. **横截面 z-score 标准化**：每天全市场拉成均值 0、标准差 1。
2. **证监会行业中性化**：每天每只股票减「所在行业均值」，剔除行业 β 干扰，只留个股相对行业的超跌部分。
3. 再按 **IS 期 IC 符号自动翻转方向**、按 **IS-ICIR 给权重**进入合成。

### 5.4 权重与历史表现

- 反转族是 Frozen 组合里**权重最大的一族（约 37%）**；单因子 `rev_5` 的 IS-ICIR 高达 **+6.37**，是全组合权重最大的单个因子（约 12%）。
- 在 2020–2023 跨周期检验中，反转族 **4/6 活**（rev_5/20/60/intraday 全活，mom_5/mom_20 失效），是少数在新冠熔断 + 2022 熊市里依然有效的族。
- 道理直观：**恐慌、暴跌、熔断时，恰恰是超跌反弹最猛时**，反转因子吃的正好是这块肉；而动量因子在那种环境会死（追涨杀跌反噬），反转反而加分。

## 7. 核心结论与约束

1. **因子有寿命 → 选股层冻结 IS 胜者**（已证优于动态门控）；状态正交因子（β/低波/困境）在截面选股层被证伪无效，不进系统。
2. **市场有状态 → 配置层用 regime，但信号尺度须与 alpha 再平衡尺度匹配**：ETF 轮动(C) 在「资产级」用 120 日宽基趋势尺度匹配故有效；本选股组合(B, 5 日再平衡)配 **20 日趋势 / 滚动波动率状态** 才有效（等权 MA20、Vol 双双优于纯股票 B）；配 C 的 120 日或等权 200 日宽基趋势则偏慢、反而更差。**这是本次复盘新增的核心约束。**
3. **跨周期证据强于单一切点**：2020–2023（含熔断+2022 熊）检验显示质量族全活、反转族 4/6 活、策略跑赢基准；相对「2024-09 后纯牛市」的绝对收益数字（被牛市抬高，不可信），跨周期相对表现更可信。
4. **权重结构**：Frozen 组合里反转族权重最大（≈37%），其次正交（≈23%）、质量（≈22%）、流动性（≈15%）、波动（≈3%）、动量（≈0.15%）；单因子 `rev_5` IS-ICIR +6.37 为最大权重因子。

## 8. 局限与下一步

- **单一 OOS regime**：仅一个干净切点（2024-09），且 OOS 仅约 1.75 年，属提示性而非结论性；理想应跨多个 regime 切换验证（需更长含退市历史面板）。
- **生存者偏差**：面板为快照（1803 只），绝对夏普/年化虚高；IS/OOS 相对排序不受影响，但绝对数字须在去偏差后面重算。
- **中性化阻塞**：当前面板无行业/市值字段外的完善中性化通道已部分实现（行业中性化已做）；规模中性化仅在质量因子上做。进一步去伪 alpha 需含 industry/mkt_cap 的面板。
- **下一步可做**：① 风险预算（vol-target / 回撤止损 / 全因子死亡转现金）；② 把配置闸从 120 日宽基信号换成 MA20/波动率状态喂给生产引擎；③ 量化「定期重冻」（如年度滚动窗口）相对一次性冻结的增益；④ 单测 2022 纯熊年份。
