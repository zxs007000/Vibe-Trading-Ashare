# 逻辑链叙事 (Logic Chain Narratives)

> 8 条逻辑链的经济学故事。每条链的期望市场时间**由叙事决定,非外部赋予**。

---

## 1. value_qlowvol — 价值质量低波

```
value → quality → volatility
```

### 投资逻辑
这是一条**买入并持有的底仓逻辑**。它的叙事是:找到市场上最便宜的股票→筛掉质量差的→用低波动避免回撤。

**节点 1: value** — "这股票便宜吗?"
用长期反转(252日或更长)识别被过度低估的资产。便宜不一定好——被低估可能是因为基本面真的在恶化。

**节点 2: quality** — "便宜但值得持有吗?"
用质量信号(量缩→投资保守、低波→盈利稳定)筛掉"价值陷阱"。只有量价行为稳定的便宜股才通过。

**节点 3: volatility** — "持有过程安全吗?"
进入后持续监控波动。如果波动突然放大,说明市场发现了我们没看到的风险,降仓或退出。

### 为什么这三段按这个顺序
不可能先看波动再找价值——波动信息不区分"便宜的高波"和"贵的高波"。必须是:估值筛选→质量去噪→波动守护。这是防御型买入的逻辑链条。

### 期望市场时间: **60 天+**
value 信号基于 252 日价格行为,天然是慢信号。quality 的波动稳定性需要 60 日窗口。这条链不追求快速止盈,它追求"买对并持有"。  
> 时间是叙事决定的:不是因为我想做长期所以选这条链,而是这条链讲了一个需要时间验证的故事。

### 什么会打破这条链
- **价值因子性价比消失**:当长期反转 IC 趋零(所有股票都在动量而非反转状态),第一层筛选失效
- **低波不低回撤**:波动率聚集——市场系统性暴跌时所有股票同时高波,第三层保护失效
- **质量→价值负传导**:便宜股的质量信号变差,穿过第一层的全是垃圾

## 2. value_momentum — 价值动量

```
value → momentum → volume
```

### 投资逻辑
价值告诉你"买什么便宜",动量告诉你"什么时候买"。两者互补——光有价值可能长期横盘,光有动量可能高位接盘。

**节点 1: value** — 圈定便宜池(同 value_qlowvol)

**节点 2: momentum** — "便宜货开始涨了吗?"
在便宜池里找近期有上涨动量的品种。这是"价值+催化剂"的逻辑——好公司低估后,如果没有资金推动,可能继续低估。

**节点 3: volume** — "这波动量有量能支撑吗?"
成交量确认趋势的真实性。无量上涨可能是技术性(大盘带动),放量上涨才是资金驱动。

### 为什么这三段按这个顺序
价值先圈池→动量在池内择时→量确认。不能动量先于价值(会选到高位追涨的),也不能量先于动量(量能信号太嘈杂)。

### 期望市场时间: **20-60 天**
value 慢,动量中速。进入条件需同时满足三个节点(=进场次数少),但一旦进场趋势可持续 1-3 个月。

### 什么会打破这条链
- **动量→价值的传导断裂**:便宜池里的动量信号不再预测未来收益,即"价值不再轮动"
- **量能噪音化**:频繁的放量但不持续,第三层反复触发假信号

## 3. value_stable — 价值稳定

```
value → volatility → liquidity
```

### 投资逻辑
一条"稳健底仓"逻辑。比 value_qlowvol 更保守——它在 quality 的位置用了 volatility,在最后的防御层加了 liquidity(确保想退就退)。

**节点 1: value** — 便宜

**节点 2: volatility** — "便宜但没有剧烈波动"
优选低波动中的便宜股。高波动便宜股可能是短期困境/游资炒作,排除。

**节点 3: liquidity** — "我想走的时候走得掉吗?"
用 Amihud 指标确保流动性足够。这是对"低波陷阱"的免疫——有些低波股是因为没有人交易,不是真的稳定。

### 为什么波动后加流动性
低波只是"不震",但"不震"的原因可能是无人问津。流动性是最后的验证——既有低波稳定性,又有充足交易深度的便宜股,才真正值得长持。

### 期望市场时间: **60 天+**
更保守的配置逻辑,不适合频繁交易。

### 什么会打破这条链
- 便宜但低波的股票恰恰是"僵尸股"——第三层流动性过滤会排除,导致无股可选
- 牛市急涨期价值因子整体失效(=所有股都贵了)

## 4. quality_momentum — 质量动量

```
quality → momentum → volume
```

### 投资逻辑
只在高性价比的股票里做趋势。质量第一层过滤掉垃圾股,动量在好公司里择时,量能确认。

**节点 1: quality** — "这公司量价行为健康吗?"
用量缩/低波/小市值信号圈定"非投机型"标的。

**节点 2: momentum** — "健康股里谁的动能最强?"
在好标的池中选动量领先的。

**节点 3: volume** — "量是实质性的吗?"
确认不是偶然的技术性动量。

### 为什么 momentum 在 quality 之后
质量第一层是非常关键的去噪——A股中有大量投机型动量(游资票的短爆),如果先做动量再筛质量,会被假信号淹没。

### 期望市场时间: **5-60 天**
quality 的中速(60d低波评估) + momentum 的快/中速。适合月度调仓。

### 什么会打破这条链
- 质量因子与动量开始正相关→两条正交性消失,链的增量价值:趋零
- 量能信号在高质量股中不起作用(=高质量股天然流动性差)

## 5. reversal_momentum — 反转接力

```
reversal → momentum → volume
```

### 投资逻辑
捕捉"跌透之后启动"的节奏。反转信号告诉你"可能到底了",但可能只是死猫跳。动量接力确认"真的起来了",量能最终确认。

**节点 1: reversal** — "超跌了吗?"
用中短窗口(20-60d)反转信号识别超跌。

**节点 2: momentum** — "超跌后开始涨了吗?"
反转信号触发后等待动量确认——如果反转后没有动量跟进,可能是噪音(单日反弹后继续跌)。

**节点 3: volume** — "涨势扎实吗?"
量确认这是资金驱动的反转,不是技术反弹。

### 为什么反转后必须接动量
反转信号单独用是高危的(抄底),动量单独用是追涨(可能追在顶)。两者接力才是"底部确认后入场"的稳健法。

### 期望市场时间: **5-20 天**
反转信号快(短窗口),动量确认需要 5-10 日。

### 什么会打破这条链
- 反转信号越做越短(市场反转节奏加快,日内就能完成)→动量确认来不及→链失效
- 下跌市中反转+动量都不work——两个因子同时负IC

## 6. vol_reversal — 波动回归

```
volatility → reversal
```

### 投资逻辑
最简单的均值回归逻辑。高波动→终将回归正常→做反转。

**节点 1: volatility** — "现在波动异常高吗?"
识别波动率异动的股票。

**节点 2: reversal** — "高波后价格在修正吗?"
确认高波动之后的回归方向。

### 为什么只有 2 段
3 段反而会削弱——在波动和反转之间加第三段是多余的,因为波动→反转的传导已经非常直接。加了量能确认反而可能"错过"(量能信号慢于反转)。

### 期望市场时间: **1-5 天**
波动聚集效应在 2-5 天窗口最显著。太短(<1天)噪音大,太长(>10天)偏离波动回归的本意。

### 什么会打破这条链
- 市场进入趋势市——波动不再回归,高波持续走高波(如2024年9月行情)
- 反转因子在高波环境中 IC 衰减——高波后不一定反转,可能继续同向

## 7. liq_momentum — 放量动量

```
liquidity → momentum → volume
```

### 投资逻辑
"量在价先"的逻辑。流动性异动(突然放量/活跃)→ 趋势启动 → 量能二次确认。

**节点 1: liquidity** — "交易活跃度突然变化?"
用 Amihud 指标或量比识别流动性跃升。

**节点 2: momentum** — "活跃度变化后价格跟上了吗?"
活跃度变化本身可能是人为造量——必须有价格跟随才算有效。

**节点 3: volume** — "持续性量能?"
确认不是一日游。持续放量+动量可持续=趋势确立。

### 为什么流动性先于动量
流动性是先行指标——人先进场,价格再涨。在 A 股散户主导的市场环境下,放量常先于价格变化 1-3 天。

### 期望市场时间: **1-5 天**
流动性信号极快(日内/1-2天),动量确认 3-5 天。

### 什么会打破这条链
- 流动性跃升不再预测后续动量→市场结构变了(比如量化占比上升,量能信号的预测力被套利掉)
- 高频假放量(程序化交易刷量)→第一层产生大量假阳性

## 8. micro_reversal — 微观反转

```
microstructure → reversal
```

### 投资逻辑
市场微观结构层面的异常(订单流不平衡/大单异动) → 短期价格修正。

**节点 1: microstructure** — "订单层面有异常吗?"
cord 系列(订单相关性)、klen/klow 系列(K线形态)捕捉微观异常。

**节点 2: reversal** — "异常后价格在修正吗?"
确认微观异常方向与反转方向一致。

### 为什么只有 2 段
microstructure 本身已经是高信噪比的信号(直接看订单流),加第三段反而引入延迟。

### 期望市场时间: **1-5 天**
订单流信号以分钟/小时为半衰期,但 A 股 T+1 制度下需将信号延至 1-5 天。

### 什么会打破这条链
- **缺乏真实的 microstructure 数据**:当前因子虽是 microstructure 标签,但 qlib158 的 cord/klen 系列本质上是基于日频 K线的形态因子,不是 L3 逐笔数据。A 股 microstructure 策略的黄金标准是 Level-2 逐笔/订单簿数据。没有 L3 数据,这条链等于"用日线K线冒充 microstructure"——可能不是真正的微观结构 alpha。
- **T+1 制度**:如果第1天信号触发,第2天才能卖出,1天延迟可能吃掉大部分 alpha

> 我们要做的不是"把链分类到短中长",而是**理解每条链的内在节律**。

## 9. index_guard_value — 指数护盾价值

```
index_guard → deep_value → macd_gate
```

### 投资逻辑
这是唯一一条**带大盘择时的逻辑链**。它的核心思想非常简单:只有在市场处于上升趋势(年线之上)时,才去寻找"便宜但有机会"的个股;一旦大盘跌破年线,不管个股多好,全撤。

**节点 1: index_guard** — "现在能买吗?"
大盘指数在 250 日均线(年线)上方 = 市场处于中长期上升趋势。年线之下 = 系统性风险,不参与。这是 A 股最重要的风控纪律——2015/2018/2022 三次大熊市,只要遵守"年线之下不持股",能回避大部分爆亏。

**节点 2: deep_value** — "买什么?"
在年线之上的安全期内,选择低 ROE + 高现金率的股票。低 ROE 意味着公司当前盈利较差(被市场低估/边缘化),高现金率意味着公司有足够的安全垫不至于破产。两者的组合 = "困境但不会死的便宜货"——一旦市场情绪好转,这类股票最容易暴涨。

**节点 3: macd_gate** — "什么时候进?什么时候出?"
MACD 金叉确认入场时机——低位便宜股开始有资金关注。MACD 死叉或指数下穿年线时出场——前者是趋势终结,后者是系统性风险。

### 为什么这三段按这个顺序
必须先做大盘判断(index_guard)。如果跳过第一层直接选股,two 个问题:
- 熊市中便宜股会一直便宜(甚至更便宜)
- 低 ROE 在熊市是"真的差",只有在牛市中才是"被低估"

第二层选股必须在第一层的安全环境下才有意义。第三层 MACD 给的是精确进出时机——不会在"便宜但还没资金关注"时过早进场。

### 为什么不同于普通 value 链
普通 value_qlowvol / value_momentum 等链**没有大盘闸门**——它们每期都在选股,熊市中也满仓。index_guard_value 则在年线之下**完全空仓**,年线之上才选股。

这意味着:
- 它的胜率可能更高(只在有利环境下交易)
- 它的持仓时间占比低(比如仅 20-30% 的交易日)
- 它的 Sharpe 可能更好(回避了熊市的回撤)
- 但它可能错过"指数在年线附近反复震荡"的锯齿行情

### 期望市场时间: **20-120 天**
年线判断是慢的(250日窗口),MACD 是中速的(12/26/9)。一次完整交易从大盘转牛到个股 MACD 死叉,通常持续 1-6 个月。

### 什么会打破这条链
- **年线反复穿越**:大盘在年线附近反复上下(如 2023 年大部分时间),频繁触发空仓→满仓→空仓,交易成本+踏空风险叠加
- **低 ROE 高现金的股票在牛市中不涨**:市场风格转向质量/成长(如 2020 年的核心资产行情),便宜货被忽略
- **MACD 金叉与年线信号重叠导致信号过少**:年线上方 + 低 ROE 高现金 + MACD 金叉,三个条件同时满足的天数可能不到 30 天/年

### 回测表现(合成数据,1 只个股)
Sharpe 0.834, AnnRet 9.94%, MaxDD -5.35%, 5 笔交易, 60% 胜率, 持仓时间 ~22%。
对比:个股买入持有 Sharpe -0.100, MaxDD -21.05%。
指数护盾在合成环境中成功回避了主要回撤,同时保持正收益。
